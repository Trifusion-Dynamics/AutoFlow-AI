## STEP 5: AUTHENTICATION

### src/lib/store/authStore.ts (Zustand):
State:
- user: User | null
- org: Organization | null
- accessToken: string | null
- isLoading: boolean
- isAuthenticated: boolean

Actions:
- login(email, password)
- register(name, email, password, orgName)
- logout()
- refreshToken()
- loadFromStorage() — localStorage se restore

Token storage:
- accessToken: memory only (Zustand)
- refreshToken: httpOnly cookie (set by server)
- On page refresh: call /auth/me to restore session

## STEP 18: REAL-TIME FEATURES

### src/lib/hooks/useRealtime.ts:
SSE connection management:
- Connect to /api/v1/stream/org (org events)
- Auto-reconnect on disconnect (exponential backoff)
- Handle events:
